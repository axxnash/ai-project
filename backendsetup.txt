STEP 1: OPEN TERMINAL
Open PowerShell / Command Prompt

STEP 2: GO TO BACKEND FOLDER
cd \backend

You should see a folder named "app"

STEP 3: CREATE VIRTUAL ENVIRONMENT
python -m venv .venv

STEP 4: ACTIVATE VIRTUAL ENVIRONMENT (WINDOWS)
.venv\Scripts\Activate.ps1

If activation is blocked, run once:
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

Then activate again:
.venv\Scripts\Activate.ps1

You should now see:
(.venv)

STEP 5: INSTALL DEPENDENCIES

If requirements.txt exists:
pip install -r requirements.txt

If not, install manually:
pip install fastapi uvicorn sqlalchemy pydantic-settings python-jose passlib[bcrypt] google-genai

STEP 6: INSTALL MYSQL DRIVER

If DATABASE_URL uses pymysql:
pip install pymysql

If DATABASE_URL uses mysqlconnector:
pip install mysql-connector-python

STEP 7: CREATE MYSQL DATABASE

Open MySQL Workbench / MySQL CLI and run:
CREATE DATABASE upm_events;

STEP 8: CREATE ENV FILE

Create a file named:
backend.env

Paste the following (edit values):

DATABASE_URL=mysql+pymysql://USERNAME:PASSWORD@localhost:3306/upm_events
JWT_SECRET=your_jwt_secret_here
JWT_ALGO=HS256
ACCESS_TOKEN_MINUTES=60
GEMINI_API_KEY=your_gemini_api_key_here

IMPORTANT:

Do NOT commit .env to GitHub

Database must already exist

STEP 9: RUN BACKEND SERVER

If main file is in backend/app/main.py:
uvicorn app.main:app --reload --port 5000

If main file is in backend/main.py:
uvicorn main:app --reload --port 5000

You should see:
Uvicorn running on http://127.0.0.1:5000

STEP 10: TEST BACKEND

Open browser:
http://localhost:5000

http://localhost:5000/health

http://localhost:5000/docs

Expected health response:
{ "status": "ok" }
