AI PROJECT 2 – FRONTEND SETUP (REACT + VITE + TAILWIND)

STEP 1: OPEN TERMINAL
Open PowerShell / Command Prompt

STEP 2: GO TO FRONTEND FOLDER
cd \frontend

You should see:

package.json

src/

index.html

STEP 3: INSTALL DEPENDENCIES
npm install

STEP 4: INSTALL REQUIRED LIBRARIES
npm install react-router-dom axios lucide-react

STEP 5: CREATE FRONTEND ENV FILE

Create a file named:
upm-events-frontend.env

Paste:
VITE_API_BASE=http://localhost:5000

(If your backend runs on a different port, change it here.)

STEP 6: CONFIRM TAILWIND IS LOADING

A) Check src\index.css contains ONLY:
@tailwind base;
@tailwind components;
@tailwind utilities;

B) Check src\main.jsx includes:
import "./index.css";

STEP 7: RUN FRONTEND DEV SERVER
npm run dev

You will see a URL like:
http://localhost:5173/

Open it in your browser.

STEP 8: QUICK TEST (OPTIONAL)
In App.jsx, put:

<div className="min-h-screen bg-slate-100 flex items-center justify-center"> <h1 className="text-4xl font-bold text-slate-900">Frontend OK ✅</h1> </div>

If you see the styling, Tailwind is working.
